- index.html → milk-collection.html (Farmer milk collection)
- pos-demo.html → pos-billing.html (Customer POS billing)
- homepage.html → index.html (Main homepage)

### Settings & Configuration
- settings.html → shop-settings.html (create if needed)

### Authentication (Optional)
- login.html → login.html

### Plan Pages
- plan-basic.html → pricing-trial.html

- plan-distributor.html → pricing-enterprise.html

## SEO Meta Tags Structure

Each page should have:
- Unique title (60 chars max)
- Meta description (155 chars max)
- Keywords
- Open Graph tags
- Canonical URL
