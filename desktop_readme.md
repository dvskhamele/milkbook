# MilkBook Desktop Application

## How to Install MilkBook as a Desktop Application

### Method 1: Browser PWA Installation (Recommended)
1. Open Chrome, Edge, or Firefox
2. Navigate to: https://inquisitive-semifreddo-a1c38b.netlify.app
3. Click the three dots menu (Chrome/Edge) or hamburger menu (Firefox)
4. Select "Install" or "Add to Desktop" or "Add to Home Screen"
5. Follow the prompts to install as a desktop application

### Method 2: Download Complete HTML File
1. Download the standalone application file: [Download MilkBook Desktop](https://inquisitive-semifreddo-a1c38b.netlify.app/final_complete_milkbook.html)
2. Save it to your desktop
3. Double-click to open in your default browser
4. Pin to taskbar for quick access

### Method 3: Manual Installation
1. Create a folder on your desktop named "MilkBook"
2. Download all the HTML files separately:
   - [Dashboard](https://inquisitive-semifreddo-a1c38b.netlify.app/dashboard.html)
   - [Collection](https://inquisitive-semifreddo-a1c38b.netlify.app/collection.html)
   - [Farmers](https://inquisitive-semifreddo-a1c38b.netlify.app/farmers.html)
   - [Ledger](https://inquisitive-semifreddo-a1c38b.netlify.app/ledger.html)
   - [Payments](https://inquisitive-semifreddo-a1c38b.netlify.app/payments.html)
   - [Sales](https://inquisitive-semifreddo-a1c38b.netlify.app/milk-sales.html)
   - [Inventory](https://inquisitive-semifreddo-a1c38b.netlify.app/inventory.html)
   - [Reports](https://inquisitive-semifreddo-a1c38b.netlify.app/reports.html)
   - [Backup](https://inquisitive-semifreddo-a1c38b.netlify.app/backup.html)
   - [Settings](https://inquisitive-semifreddo-a1c38b.netlify.app/settings.html)
   - [Logout](https://inquisitive-semifreddo-a1c38b.netlify.app/logout.html)
3. Save all files in the MilkBook folder
4. Open index.html in your browser to start the application

## Desktop Features:
- Works completely offline after first load
- All data stored locally in your browser
- No internet required for daily operations
- Syncs to cloud when internet is available
- Full dairy management functionality
- Farmer evidence system
- Payment tracking
- Reports and analytics

## System Requirements:
- Windows 7+, macOS 10.12+, Linux
- Chrome, Edge, Firefox, or Safari browser
- At least 500MB free disk space
- 2GB RAM recommended

## Data Safety:
- All data stored in your browser's local storage
- Regular backups recommended
- Export data using the Backup section
- Data remains on your device unless synced to cloud
